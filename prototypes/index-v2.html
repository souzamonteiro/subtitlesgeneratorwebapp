How this works:

  • Upload an audio file containing speech (MP3, WAV, etc.)
  • The app will play the audio and use the Web Speech API to transcribe it
  • You need to allow microphone permissions for the speech recognition to work
  • The recognition happens in real-time as the audio plays
  • For long files, processing may take time and accuracy varies

This application uses the Web Speech API for real speech recognition.

Audio processing happens client-side - no data is sent to external servers.
